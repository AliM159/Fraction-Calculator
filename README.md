## Getting Started

### Option 1: Run the Executable
An executable version of the calculator (`FractionCalculator.exe`) is included in the repository.

1. Download or clone this repository.
2. Navigate to the `bin/Release` or `bin/Debug` folder.
3. Double-click `FractionCalculator.exe` to launch the application.

> ⚠️ **Note:** Some systems may block `.exe` files downloaded from the internet.  
> If prompted, right-click → “Properties” → “Unblock” before running.

### Option 2: Open in Visual Studio
If you prefer to view or modify the code:

1. Open the solution (`.sln`) or project file in **Visual Studio**.
2. Build the project:
   - `Build` → `Build Solution`
3. Run the application:
   - Press `F5` or click the **Start** button.
